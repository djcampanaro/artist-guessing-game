Artist Guessing Game

Artist Guessing Game is built solely with html, css, and javascript. All artworks are sourced from the Art Institute of Chicago's API. 
The challenge is to pick the correct artist amongst three choices for each image. A counter in the top right keeps track of the number of correct answers.

Instructions:

-Clone the repository

-Open the index.html file in a browser (built using Safari)

-Try your hand at the artist guessing game
